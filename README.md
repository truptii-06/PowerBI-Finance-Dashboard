# 📊 PowerBI-Finance-Dashboard

## 📌 Project Overview

This project presents an interactive financial dashboard built using Power BI to analyze key business metrics such as revenue, profit, and cost trends. The dashboard enables users to gain meaningful insights and make data-driven decisions.

---

## 🎯 Objectives

* Analyze revenue trends over time
* Identify high-performing financial segments
* Understand profit distribution across categories
* Detect cost-heavy areas impacting profitability

---

## 🛠 Tools & Technologies

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation

---

## 📂 Dataset

* Financial dataset containing sales, profit, and cost information
* Format: Excel (.xlsx)
* File included: `Financial Sample.xlsx`

---

## 📈 Dashboard Features

* 📊 Revenue trend analysis (time-based)
* 💰 Profit breakdown by category
* 📉 Cost distribution visualization
* 🎛 Interactive filters and slicers
* 📌 KPI cards for quick insights

---

## 🔍 Key Insights

* Revenue shows consistent growth with periodic fluctuations
* Certain categories contribute significantly higher profit margins
* High operational costs in specific areas reduce overall profitability
* Seasonal trends affect revenue performance
